API section

When you need to work with API request in angular, first thing first you need to import HttpClientModule

import {{ '{' }} HttpClientModule {{ '}' }} from '@angular/common/http'

and in the services you need to import HttpClient

import {{ "{" }} HttpClient {{ '}' }} from '@angular/common/http';

ang injected into constructor

constructor(private http: HttpClient)

The example

fetchUser() {{ '{' }} return this.http.get('https://jsonplaceholder.typicode.com/users') {{ '}'}}
