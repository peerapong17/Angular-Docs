Router

To use router in angular. First of all you should import routes and routermodule

import {{"{"}} Routes, RouterModule {{"}"}} from '@angular/router';

and create whatever path you want

and if you want to use router to push user to another page or get parameter from url

import {{ '{' }} ActivatedRoute,Router {{ '}' }} from '@angular/router';

and injected into constructor

constructor(private route: ActivatedRoute, private router: Router)

The example of using ActivatedRoute

let id = route.snapshot.params['id']

The example of using Router

this.router.navigateByUrl('/')
